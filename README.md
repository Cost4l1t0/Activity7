# Activity 7 - Robotics Management System

## Project Description
This system manages a robotics school infrastructure. It includes user authentication with specific roles (Administrative, Teacher, Student), management of robotics kits assigned to users, and a catalog of courses and groups. Built with Laravel 11.

## Setup
1. Clone the repository
2. Run `composer install`
3. Configure `.env` file
4. Run `php artisan migrate --seed`
